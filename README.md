# Football Analytics Dashboard

An end-to-end football analytics dashboard built using Palantir Foundry, SQL, and interactive data visualizations.

## Demo
[Watch the Dashboard Demo](https://youtu.be/GHwQzAdxrZw?si=0k8LRePr33s6q8Yk)

## Overview
Built a full analytics pipeline and interactive dashboard to visualize football player and team performance metrics. The project covers the full data lifecycle — from raw ingestion to cleaned, structured data powering a live dashboard.

## What I Built
- **Data Pipelines** — Designed and built pipelines in Palantir Foundry to ingest and process raw football performance data
- **Data Cleaning** — Standardized and validated datasets to ensure accuracy and consistency across sources
- **SQL Transforms** — Wrote SQL queries to structure, filter, and aggregate data for dashboard consumption
- **Interactive Dashboard** — Built visualizations to surface player and team metrics in a clear, actionable format

## Tech Stack
- Palantir Foundry
- SQL
- Data Pipeline Design
- Dashboard & Visualization Tools
